# Имаджинариум

Здесь вы можете разместить изображения карточек и рубашку для новой игры "Имаджинариум".

Структура:
- `cards/` — сюда кладите все карты, которые будут выбирать игроки.
- `back/` — сюда кладите изображение рубашки одной карты.

Формат файлов: `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`.

В приложении при старте игры карточки будут загружаться из этой папки автоматически.
